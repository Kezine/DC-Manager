Barre d'onglets (desktop, icônes seules)

Sur desktop, chaque onglet est une ICÔNE seule (libellé porté par title + aria-label). L'onglet actif est souligné (.tab.active::after). Le badge de comptage est un .tab-count.

Onglet groupe (déroulant, .tab-group)

Un onglet .tab suivi d'un caret .tabs-dd-caret, avec un menu .tabs-dd-menu (ouvert ici en statique).

Menu déroulant responsive (.tabs-dd)

Sur petit écran, la barre d'onglets cède la place à ce déclencheur unique (.tabs-dd-trigger) + menu global (groupes aplatis). Forcé visible ici (masqué > 760 px dans l'app).
